Background
The National Highway Traffic Safety Administration (NHTSA) is a federal agency that aims to minimize vehicular incidents and potential harm from the incidents. The NHTSA publishes multi-year datasets with a multitude of information ranging from age of the driver to weather conditions during an incident.

Data

Data was downloaded from the Crash Reporting Sampling System website. The data was a random sampling from millions of reported crashes each year, each dataset had over 50,000 accidents. All Gas No Brakes focused on three datasets pertaining to accidents, vehicles, and people.

The three datasets were loaded into a PostgreSQL database hosted in AWS RDS (Amazon Web Services Relational Database Service). The database is accessible to both the web application and Python for creating machine learning models.

Before the data could be put through a machine learning model, it was preprocessed to remove fields that did not contribute to an accident. For example, one field that was removed before running the model pertained to the need of transportation to a hospital for a person who had been in an accident. This field was more of a result of an accident instead of a determinant. A total of 30 factors were used to create the machine learning model that gave three outcomes (any injury, a serious injury, or a fatal injury).

Once the machine learning model was finished, a Node.js web application was built and is currently running on this Heroku platform.


About All Gas No Brakes
All Gas No Brakes is a composed of five students from the University of Minnesota Data Analytics and Visualizations bootcamp. We created this website in an effort to let users interact with our interface and leverage the machine learning model we created to predict severity of an injury in the result of a crash.
